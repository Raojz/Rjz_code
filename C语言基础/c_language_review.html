第一部分：基础语法

1. 变量与数据类型

1.1 基本数据类型

C语言提供了多种基本数据类型，包括：

  • 整型：int, short, long, long long
  • 浮点型：float, double
  • 字符型：char
  • 布尔型：_Bool（C99标准）

语法示例：

int age = 25;
float height = 1.75f;
char grade = 'A';
_Bool is_student = 1;

常见应用场景：

  • 整型用于计数、索引等整数运算
  • 浮点型用于科学计算、金融计算等需要高精度的场景
  • 字符型用于存储单个字符或ASCII码
注意事项：
- 不同数据类型占用的内存大小可能因平台而异
- 浮点型存在精度误差，不宜用于精确比较

1.2 变量声明与初始化

变量声明的语法为：数据类型 变量名;

变量初始化的语法为：数据类型 变量名 = 初始值;

语法示例：

// 声明变量
int x;
// 初始化变量
x = 10;
// 声明并初始化变量
int y = 20;

常见应用场景：

  • 在函数开头声明所有变量（传统C风格）
  • 在需要时声明变量（C99及以后标准）
注意事项：
- 未初始化的局部变量值是未定义的
- 全局变量默认初始化为0

2. 运算符与表达式

2.1 算术运算符

C语言提供了以下算术运算符：

  • +：加法
  • -：减法
  • *：乘法
  • /：除法
  • %：取模

语法示例：

int a = 10, b = 3;
int sum = a + b;      // 13
int diff = a - b;     // 7
int product = a * b;  // 30
int quotient = a / b; // 3（整数除法）
int remainder = a % b;// 1

常见应用场景：

  • 基本数学运算
  • 循环计数
注意事项：
- 整数除法会截断小数部分
- 取模运算符的结果符号与被除数相同

2.2 关系运算符

关系运算符用于比较两个值的大小关系：

  • ==：等于
  • !=：不等于
  • >：大于
  • <：小于
  • >=：大于等于
  • <=：小于等于

语法示例：

int a = 10, b = 20;
_Bool is_equal = (a == b);    // 0（假）
_Bool is_greater = (a > b);   // 0（假）
_Bool is_less = (a < b);      // 1（真）

常见应用场景：

  • 条件判断
  • 循环终止条件
注意事项：
- 不要将赋值运算符=与相等运算符==混淆
- 关系运算符的结果是布尔值（0或1）

2.3 逻辑运算符

逻辑运算符用于组合多个条件：

  • &&：逻辑与
  • ||：逻辑或
  • !：逻辑非

语法示例：

int age = 25;
_Bool is_adult = (age >= 18);
_Bool is_student = (age <= 22);
_Bool is_adult_student = is_adult && is_student; // 1（真）

常见应用场景：

  • 复杂条件判断
  • 错误处理
注意事项：
- 逻辑与和逻辑或具有短路求值特性
- 逻辑非运算符的优先级较高

3. 控制流语句

3.1 if-else语句

if-else语句用于根据条件执行不同的代码块。

语法示例：

int score = 85;
if (score >= 90) {
    printf("优秀\n");
} else if (score >= 80) {
    printf("良好\n");
} else if (score >= 70) {
    printf("中等\n");
} else {
    printf("不及格\n");
}

常见应用场景：

  • 分支选择
  • 错误处理
注意事项：
- 条件表达式必须是布尔值
- 大括号可以省略，但不建议（容易引入bug）

3.2 switch-case语句

switch-case语句用于多分支选择。

语法示例：

int day = 3;
switch (day) {
    case 1:
        printf("星期一\n");
        break;
    case 2:
        printf("星期二\n");
        break;
    case 3:
        printf("星期三\n");
        break;
    default:
        printf("无效日期\n");
}

常见应用场景：

  • 菜单选择
  • 状态机实现
注意事项：
- case标签必须是常量表达式
- 每个case分支末尾需要添加break语句，否则会继续执行下一个case

3.3 循环语句

C语言提供了三种循环语句：for、while和do-while。

for循环语法示例：

for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}

while循环语法示例：

int i = 0;
while (i < 10) {
    printf("%d\n", i);
    i++;
}

do-while循环语法示例：

int i = 0;
do {
    printf("%d\n", i);
    i++;
} while (i < 10);

常见应用场景：

  • 遍历数组
  • 重复执行某个操作
  • 输入验证
注意事项：
- for循环适合已知循环次数的场景
- while循环适合未知循环次数的场景
- do-while循环至少执行一次

第二部分：函数与指针

4. 函数定义与调用

4.1 函数定义

函数定义的语法为：

返回值类型 函数名(参数列表) {
    // 函数体
    return 返回值;
}

语法示例：

int add(int a, int b) {
    return a + b;
}

常见应用场景：

  • 代码复用
  • 模块化设计
注意事项：
- 函数名应具有描述性
- 参数列表中的参数需要指定类型

4.2 函数调用

函数调用的语法为：函数名(参数列表);

语法示例：

int result = add(10, 20);
printf("%d\n", result); // 输出30

常见应用场景：

  • 执行特定功能
  • 计算结果
注意事项：
- 实参的数量和类型必须与形参匹配
- 函数调用会创建栈帧，存储局部变量和返回地址

4.3 函数原型

函数原型用于声明函数的接口，语法为：返回值类型 函数名(参数列表);

语法示例：

int add(int a, int b); // 函数原型

常见应用场景：

  • 头文件中声明函数接口
  • 解决函数调用顺序问题
注意事项：
- 函数原型必须与函数定义一致
- 参数名称可以省略，但建议保留以提高可读性

5. 指针基础

5.1 指针声明

指针声明的语法为：数据类型 *指针变量名;

语法示例：

int x = 10;
int *p = &x; // p指向x的地址

常见应用场景：

  • 动态内存分配
  • 函数参数传递（传址调用）
  • 数组操作
注意事项：
- 未初始化的指针是野指针，可能导致程序崩溃
- 空指针（NULL）不指向任何内存地址

5.2 指针操作

指针的基本操作包括：

  • &：取地址运算符
  • *：解引用运算符

语法示例：

int x = 10;
int *p = &x; // p指向x的地址
printf("x的值：%d\n", x);    // 输出10
printf("x的地址：%p\n", &x); // 输出x的地址
printf("p的值：%p\n", p);   // 输出x的地址
printf("p指向的值：%d\n", *p); // 输出10

常见应用场景：

  • 修改函数外部变量
  • 遍历数组
  • 动态数据结构（链表、树等）
注意事项：
- 解引用空指针会导致程序崩溃
- 指针类型必须与指向的数据类型匹配

6. 函数指针与回调

6.1 函数指针声明

函数指针声明的语法为：返回值类型 (*指针变量名)(参数列表);

语法示例：

int add(int a, int b) {
    return a + b;
}

int (*fp)(int, int) = add; // fp指向add函数

常见应用场景：

  • 回调函数
  • 动态函数调用
  • 状态机实现
注意事项：
- 函数指针的类型必须与指向的函数类型匹配
- 函数名本身就是函数的地址

6.2 回调函数

回调函数是指通过函数指针调用的函数。

语法示例：

// 回调函数类型定义
typedef int (*Callback)(int, int);

// 执行回调的函数
int execute_callback(Callback cb, int a, int b) {
    return cb(a, b);
}

// 回调函数实现
int multiply(int a, int b) {
    return a * b;
}

int main() {
    int result = execute_callback(multiply, 10, 20);
    printf("%d\n", result); // 输出200
    return 0;
}

常见应用场景：

  • 事件驱动编程
  • 排序算法中的比较函数
  • 异步操作完成后的处理
注意事项：
- 回调函数的参数和返回值类型必须与函数指针类型匹配
- 回调函数可以是静态函数或全局函数

第三部分：数组与字符串

7. 数组操作

7.1 数组声明

数组声明的语法为：数据类型 数组名[数组大小];

语法示例：

int numbers[5]; // 声明一个包含5个整数的数组
int scores[] = {90, 85, 95, 80, 75}; // 声明并初始化数组

常见应用场景：

  • 存储一组相同类型的数据
  • 遍历和排序
注意事项：
- 数组大小必须是常量表达式
- 数组下标从0开始

7.2 数组访问

数组访问的语法为：数组名[下标];

语法示例：

int scores[] = {90, 85, 95, 80, 75};
printf("第一个元素：%d\n", scores[0]); // 输出90
printf("最后一个元素：%d\n", scores[4]); // 输出75

常见应用场景：

  • 遍历数组元素
  • 修改数组元素
注意事项：
- 数组下标不能越界，否则会导致未定义行为
- 数组名是常量指针，不能修改

7.3 多维数组

多维数组声明的语法为：数据类型 数组名[第一维大小][第二维大小];

语法示例：

int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

常见应用场景：

  • 矩阵运算
  • 图像存储
  • 表格数据
注意事项：
- 多维数组在内存中是按行存储的
- 多维数组的访问需要指定所有维度的下标

8. 字符串处理

8.1 字符串表示

C语言中，字符串是以\0（空字符）结尾的字符数组。

语法示例：

char str1[] = "Hello"; // 自动添加\0
char str2[6] = {'H', 'e', 'l', 'l', 'o', '\0'};

常见应用场景：

  • 文本处理
  • 用户输入
  • 文件名处理
注意事项：
- 字符串长度不包括\0
- 字符串数组的大小必须比实际字符串长度大1

8.2 字符串函数

C标准库提供了丰富的字符串处理函数，定义在<string.h>头文件中。

常用字符串函数：

  • strlen：计算字符串长度
  • strcpy：复制字符串
  • strcat：拼接字符串
  • strcmp：比较字符串
  • strchr：查找字符
  • strstr：查找子串

语法示例：

#include 

char str1[20] = "Hello";
char str2[] = "World";

// 计算字符串长度
printf("str1长度：%zu\n", strlen(str1)); // 输出5

// 复制字符串
strcpy(str1, str2);
printf("str1：%s\n", str1); // 输出World

// 拼接字符串
strcat(str1, "!");
printf("str1：%s\n", str1); // 输出World!

// 比较字符串
int result = strcmp(str1, str2);
printf("比较结果：%d\n", result); // 输出非0

常见应用场景：

  • 字符串操作
  • 输入验证
  • 文本解析
注意事项：
- strcpy和strcat可能导致缓冲区溢出，建议使用strncpy和strncat
- 字符串比较是按ASCII码值进行的

第四部分：结构体与联合体

9. 结构体定义与使用

9.1 结构体定义

结构体定义的语法为：

struct 结构体名 {
    数据类型 成员名1;
    数据类型 成员名2;
    // ...
};

语法示例：

struct Person {
    char name[20];
    int age;
    float height;
};

常见应用场景：

  • 存储复杂数据类型
  • 表示实体（如用户、商品等）
  • 网络数据包
注意事项：
- 结构体成员可以是任意数据类型
- 结构体大小是成员大小的总和（可能有内存对齐）

9.2 结构体变量声明与初始化

结构体变量声明的语法为：struct 结构体名 变量名;

语法示例：

struct Person p1;
strcpy(p1.name, "张三");
p1.age = 25;
p1.height = 1.75f;

// 初始化结构体变量
struct Person p2 = {"李四", 30, 1.80f};

常见应用场景：

  • 创建实体对象
  • 存储相关联的数据
注意事项：
- 结构体成员访问使用.运算符
- 结构体初始化的顺序必须与成员定义顺序一致

9.3 结构体数组

结构体数组声明的语法为：struct 结构体名 数组名[数组大小];

语法示例：

struct Person people[2] = {
    {"张三", 25, 1.75f},
    {"李四", 30, 1.80f}
};

常见应用场景：

  • 存储多个实体对象
  • 数据库记录
注意事项：
- 结构体数组的访问方式与普通数组类似
- 可以通过下标访问结构体成员

10. 联合体与枚举

10.1 联合体定义

联合体定义的语法为：

union 联合体名 {
    数据类型 成员名1;
    数据类型 成员名2;
    // ...
};

语法示例：

union Data {
    int i;
    float f;
    char str[20];
};

常见应用场景：

  • 节省内存（多个成员共享同一块内存）
  • 类型转换
  • 硬件寄存器访问
注意事项：
- 联合体的大小是最大成员的大小
- 同一时间只能有一个成员有效

10.2 枚举定义

枚举定义的语法为：

enum 枚举名 {
    枚举值1,
    枚举值2,
    // ...
};

语法示例：

enum Weekday {
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday,
    Sunday
};

常见应用场景：

  • 表示有限的状态集合
  • 提高代码可读性
  • 替代魔法数字
注意事项：
- 枚举值默认从0开始递增
- 可以显式指定枚举值

11. 结构体指针与动态分配

11.1 结构体指针

结构体指针声明的语法为：struct 结构体名 *指针变量名;

语法示例：

struct Person p = {"张三", 25, 1.75f};
struct Person *pp = &p;

// 访问结构体成员
printf("姓名：%s\n", pp->name); // 输出张三
printf("年龄：%d\n", pp->age);  // 输出25

常见应用场景：

  • 动态内存分配
  • 链表节点
  • 函数参数传递（传址调用）
注意事项：
- 结构体指针访问成员使用->运算符
- 结构体指针可以指向动态分配的内存

11.2 动态分配结构体

动态分配结构体的语法为：struct 结构体名 *指针变量名 = malloc(sizeof(struct 结构体名));

语法示例：

struct Person *pp = malloc(sizeof(struct Person));
if (pp != NULL) {
    strcpy(pp->name, "张三");
    pp->age = 25;
    pp->height = 1.75f;
    free(pp); // 释放内存
}

常见应用场景：

  • 动态创建结构体对象
  • 链表、树等动态数据结构
注意事项：
- 动态分配的内存需要手动释放，否则会导致内存泄漏
- 分配失败会返回NULL，需要检查

第五部分：内存管理

12. 静态内存与动态内存

12.1 静态内存

静态内存是指在编译时分配的内存，包括：

  • 全局变量
  • 静态变量
  • 常量

语法示例：

int global_var = 10; // 全局变量

void func() {
    static int static_var = 20; // 静态变量
    static_var++;
    printf("%d\n", static_var);
}

常见应用场景：

  • 存储全局配置
  • 计数器
  • 常量数据
注意事项：
- 静态内存的生命周期是整个程序运行期间
- 静态变量只会初始化一次

12.2 动态内存

动态内存是指在运行时分配的内存，使用malloc、calloc、realloc等函数分配。

语法示例：

int *p = malloc(10 * sizeof(int)); // 分配10个整数的内存
if (p != NULL) {
    // 使用内存
    free(p); // 释放内存
}

常见应用场景：

  • 动态数组
  • 链表、树等动态数据结构
  • 大数据存储
注意事项：
- 动态内存需要手动释放，否则会导致内存泄漏
- 动态内存的生命周期由程序员控制

13. 动态内存分配

13.1 malloc函数

malloc函数用于分配指定大小的内存，语法为：void *malloc(size_t size);

语法示例：

int *p = malloc(10 * sizeof(int));
if (p != NULL) {
    for (int i = 0; i < 10; i++) {
        p[i] = i;
    }
    free(p);
}

常见应用场景：

  • 动态数组
  • 结构体动态分配
注意事项：
- malloc分配的内存是未初始化的
- 分配失败返回NULL

13.2 calloc函数

calloc函数用于分配指定数量和大小的内存，并初始化为0，语法为：void *calloc(size_t num, size_t size);

语法示例：

int *p = calloc(10, sizeof(int));
if (p != NULL) {
    // p中的元素初始化为0
    free(p);
}

常见应用场景：

  • 需要初始化为0的动态数组
  • 结构体动态分配
注意事项：
- calloc比malloc多了初始化步骤
- 分配失败返回NULL

13.3 realloc函数

realloc函数用于重新分配内存，语法为：void *realloc(void *ptr, size_t size);

语法示例：

int *p = malloc(10 * sizeof(int));
if (p != NULL) {
    // 重新分配内存为20个整数
    p = realloc(p, 20 * sizeof(int));
    if (p != NULL) {
        free(p);
    }
}

常见应用场景：

  • 动态调整数组大小
  • 扩展内存
注意事项：
- realloc可能会移动内存块
- 分配失败返回NULL，原内存块仍然有效

13.4 free函数

free函数用于释放动态分配的内存，语法为：void free(void *ptr);

语法示例：

int *p = malloc(10 * sizeof(int));
if (p != NULL) {
    free(p);
    p = NULL; // 避免野指针
}

常见应用场景：

  • 释放不再使用的动态内存
  • 防止内存泄漏
注意事项：
- 只能释放动态分配的内存
- 不能重复释放同一块内存

14. 指针运算与内存布局

14.1 指针运算

指针可以进行以下运算：

  • 加法：指针 + 整数
  • 减法：指针 - 整数
  • 比较：指针1 == 指针2、指针1 < 指针2等

语法示例：

int arr[] = {10, 20, 30, 40, 50};
int *p = arr;

printf("%d\n", *p);   // 输出10
printf("%d\n", *(p+1)); // 输出20
printf("%d\n", *(p+2)); // 输出30

常见应用场景：

  • 遍历数组
  • 动态数据结构操作
  • 内存块操作
注意事项：
- 指针运算的单位是指向的数据类型的大小
- 指针不能进行乘法和除法运算

14.2 内存布局

C程序的内存布局包括以下几个部分：

  • 代码段：存储程序的机器指令
  • 数据段：存储全局变量和静态变量
  • 堆：存储动态分配的内存
  • 栈：存储局部变量和函数调用信息

常见应用场景：

  • 理解程序运行机制
  • 调试内存问题
  • 优化内存使用
注意事项：
- 栈的大小是有限的，递归过深会导致栈溢出
- 堆的大小受限于系统内存

第六部分：文件操作

15. 文件打开与关闭

15.1 fopen函数

fopen函数用于打开文件，语法为：FILE *fopen(const char *filename, const char *mode);

常用打开模式：

  • "r"：只读模式
  • "w"：写入模式（覆盖原有文件）
  • "a"：追加模式（在文件末尾写入）
  • "r+"：读写模式
  • "w+"：读写模式（覆盖原有文件）
  • "a+"：读写模式（在文件末尾写入）

语法示例：

FILE *fp = fopen("data.txt", "r");
if (fp != NULL) {
    // 操作文件
    fclose(fp);
}

常见应用场景：

  • 读取配置文件
  • 写入日志文件
  • 数据持久化
注意事项：
- 打开失败返回NULL
- 需要检查文件是否成功打开

15.2 fclose函数

fclose函数用于关闭文件，语法为：int fclose(FILE *stream);

语法示例：

FILE *fp = fopen("data.txt", "r");
if (fp != NULL) {
    fclose(fp);
}

常见应用场景：

  • 释放文件资源
  • 确保数据写入磁盘
注意事项：
- 关闭文件后，文件指针不再有效
- 程序退出时会自动关闭所有打开的文件

16. 文件读写操作

16.1 字符读写

  • fgetc：读取一个字符
  • fputc：写入一个字符

语法示例：

FILE *fp = fopen("data.txt", "w");
if (fp != NULL) {
    fputc('H', fp);
    fputc('e', fp);
    fputc('l', fp);
    fclose(fp);
}

fp = fopen("data.txt", "r");
if (fp != NULL) {
    int c;
    while ((c = fgetc(fp)) != EOF) {
        printf("%c", c);
    }
    fclose(fp);
}

常见应用场景：

  • 文本文件处理
  • 逐字符解析
注意事项：
- fgetc返回EOF表示文件结束
- EOF是一个宏，值为-1

16.2 字符串读写

  • fgets：读取一行字符串
  • fputs：写入一行字符串

语法示例：

FILE *fp = fopen("data.txt", "w");
if (fp != NULL) {
    fputs("Hello World!\n", fp);
    fclose(fp);
}

fp = fopen("data.txt", "r");
if (fp != NULL) {
    char buf[100];
    while (fgets(buf, sizeof(buf), fp) != NULL) {
        printf("%s", buf);
    }
    fclose(fp);
}

常见应用场景：

  • 文本文件处理
  • 配置文件读取
注意事项：
- fgets会读取换行符
- fgets最多读取size-1个字符

16.3 二进制读写

  • fread：读取二进制数据
  • fwrite：写入二进制数据

语法示例：

struct Person {
    char name[20];
    int age;
    float height;
};

struct Person p = {"张三", 25, 1.75f};

FILE *fp = fopen("person.dat", "wb");
if (fp != NULL) {
    fwrite(&p, sizeof(struct Person), 1, fp);
    fclose(fp);
}

fp = fopen("person.dat", "rb");
if (fp != NULL) {
    struct Person p2;
    fread(&p2, sizeof(struct Person), 1, fp);
    printf("姓名：%s\n", p2.name);
    printf("年龄：%d\n", p2.age);
    printf("身高：%.2f\n", p2.height);
    fclose(fp);
}

常见应用场景：

  • 二进制文件处理
  • 数据持久化
  • 网络传输
注意事项：
- 二进制文件无法用文本编辑器直接查看
- 二进制读写需要保证数据类型一致

17. 文件定位与错误处理

17.1 文件定位

  • fseek：设置文件指针位置
  • ftell：获取文件指针位置
  • rewind：将文件指针重置到文件开头

语法示例：

FILE *fp = fopen("data.txt", "r");
if (fp != NULL) {
    // 将文件指针移动到第10个字节
    fseek(fp, 10, SEEK_SET);
    // 获取文件指针位置
    long pos = ftell(fp);
    printf("当前位置：%ld\n", pos);
    // 重置文件指针到开头
    rewind(fp);
    fclose(fp);
}

常见应用场景：

  • 随机访问文件
  • 大型文件处理
注意事项：
- fseek的第三个参数可以是SEEK_SET（文件开头）、SEEK_CUR（当前位置）或SEEK_END（文件末尾）
- 二进制文件和文本文件的定位方式可能不同

17.2 错误处理

  • ferror：检查文件操作是否出错
  • clearerr：清除文件错误标志

语法示例：

FILE *fp = fopen("data.txt", "r");
if (fp != NULL) {
    char buf[100];
    if (fgets(buf, sizeof(buf), fp) == NULL) {
        if (ferror(fp)) {
            printf("文件读取错误\n");
            clearerr(fp);
        }
    }
    fclose(fp);
}

常见应用场景：

  • 错误处理
  • 调试文件操作问题
注意事项：
- ferror返回非0表示出错
- clearerr用于重置错误标志

第七部分：预处理指令

18. 宏定义与条件编译

18.1 宏定义

宏定义的语法为：#define 宏名 替换文本

语法示例：

#define PI 3.1415926
#define MAX(a, b) ((a) > (b) ? (a) : (b))

float area = PI * 5 * 5;
int max_val = MAX(10, 20);

常见应用场景：

  • 定义常量
  • 定义简单函数
  • 代码复用
注意事项：
- 宏替换是文本替换，不是函数调用
- 宏参数需要加括号，避免优先级问题

18.2 条件编译

条件编译的语法为：

#ifdef 宏名
    // 代码块1
#else
    // 代码块2
#endif

语法示例：

#define DEBUG

#ifdef DEBUG
    printf("调试信息\n");
#else
    printf("发布版本\n");
#endif

常见应用场景：

  • 调试代码
  • 跨平台开发
  • 版本控制
注意事项：
- 条件编译是在编译时进行的
- 可以使用#ifndef判断宏是否未定义

19. 文件包含与头文件

19.1 文件包含

文件包含的语法为：#include <头文件> 或 #include "头文件"

语法示例：

#include  // 标准库头文件
#include "myheader.h" // 自定义头文件

常见应用场景：

  • 引入标准库函数
  • 引入自定义函数原型
  • 模块化设计
注意事项：
- <>用于标准库头文件，""用于自定义头文件
- 头文件可以嵌套包含

19.2 头文件编写

头文件通常包含：

  • 函数原型
  • 宏定义
  • 结构体、枚举定义

语法示例：

#ifndef MYHEADER_H
#define MYHEADER_H

#define PI 3.1415926

struct Person {
    char name[20];
    int age;
};

int add(int a, int b);

#endif // MYHEADER_H

常见应用场景：

  • 接口定义
  • 代码复用
  • 模块化设计
注意事项：
- 头文件需要包含保护，避免重复包含
- 头文件不应包含函数实现（除非是内联函数）

第八部分：高级主题

20. 位运算与掩码

20.1 位运算符

C语言提供了以下位运算符：

  • &：按位与
  • |：按位或
  • ^：按位异或
  • ~：按位取反
  • <<：左移
  • >>：右移

语法示例：

int a = 0b1010; // 10
int b = 0b0110; // 6

int result1 = a & b; // 0b0010（2）
int result2 = a | b; // 0b1110（14）
int result3 = a ^ b; // 0b1100（12）
int result4 = ~a;    // 按位取反
int result5 = a << 1; // 0b10100（20）
int result6 = a >> 1; // 0b0101（5）

常见应用场景：

  • 硬件控制
  • 数据压缩
  • 加密算法
  • 状态标志
注意事项：
- 位运算操作的是二进制位
- 左移相当于乘以2，右移相当于除以2

20.2 掩码

掩码是一种位运算技术，用于提取或设置特定位。

语法示例：

// 提取低8位
int value = 0x1234;
int low8 = value & 0xFF; // 0x34

// 设置第3位
int flag = 0;
flag |= (1 << 2); // 设置第3位（从0开始计数）

// 清除第3位
flag &= ~(1 << 2); // 清除第3位

常见应用场景：

  • 硬件寄存器操作
  • 状态标志处理
  • 数据打包
注意事项：
- 掩码需要根据具体需求设计
- 位运算比算术运算更快

21. 递归与栈

21.1 递归函数

递归函数是指调用自身的函数。

语法示例：

int factorial(int n) {
    if (n == 0) {
        return 1;
    }
    return n * factorial(n - 1);
}

常见应用场景：

  • 阶乘计算
  • 斐波那契数列
  • 树遍历
  • 分治算法
注意事项：
- 递归函数需要有终止条件
- 递归深度过深会导致栈溢出

21.2 栈

栈是一种后进先出（LIFO）的数据结构，C程序的栈用于存储：

  • 局部变量
  • 函数参数
  • 返回地址

语法示例：

void func(int a) {
    int b = 10;
    // 局部变量a和b存储在栈中
}

常见应用场景：

  • 函数调用
  • 表达式求值
  • 递归实现
注意事项：
- 栈的大小是有限的
- 栈溢出会导致程序崩溃

22. 错误处理与调试

22.1 错误处理

C语言的错误处理主要通过返回值和全局变量errno实现。

语法示例：

#include 

FILE *fp = fopen("data.txt", "r");
if (fp == NULL) {
    printf("文件打开失败：%s\n", strerror(errno));
    return 1;
}

常见应用场景：

  • 文件操作
  • 内存分配
  • 系统调用
注意事项：
- errno在成功调用时不会被重置
- 需要在调用前清除errno

22.2 调试

C语言的调试方法包括：

  • 打印调试（printf）
  • 调试器（如GDB、Visual Studio调试器）
  • 断言（assert）

语法示例：

#include 

int add(int a, int b) {
    assert(a >= 0 && b >= 0); // 断言a和b非负
    return a + b;
}

常见应用场景：

  • 代码调试
  • 错误检测
  • 单元测试
注意事项：
- 断言在发布版本中会被禁用
- 断言用于检查程序内部错误
